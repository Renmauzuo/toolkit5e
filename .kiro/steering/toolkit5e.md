# toolkit5e Project Steering

This workspace contains D&D 5th edition tooling packages.

## Package Structure

- **@toolkit5e/base** - Shared types and utilities used across all packages
- **@toolkit5e/monster-scaler** - Calculator for scaling monsters to different CRs

## Naming Conventions

- Workspace name: `toolkit5e`
- Package scope: `@toolkit5e`
- Package folders: `packages/<name>`

## Build Process

Run `npm run build` from the workspace root to compile all packages.
